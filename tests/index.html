How do assistive technologies handle Font Awesome?

This document contains different ways to make icon fonts—deliviered by Font Awesome on this page—accessible.

Test one: aria-hidden on the icon

We can hide the icon form assistive technologies (AT for short) by adding aria-hidden. This will however not convey the meaning—or any other information it caries—to users of AT.

Example

The link in this example has an icon of a bed before the text “Book a hotel”. The icon is however hidden for AT with aria-hidden.

Book a hotel

<a href="#book-hotel"><i class="fa fa-bed" aria-hidden="true"></i> Book a hotel</a>
